Visual Product Search Engine
New

Image-only fashion search on the DeepFashion In-Shop benchmark (300 products, 1,027 queries) hitting R@1 = 0.683 and R@10 = 0.913 with sub-millisecond search. CLIP embeddings + a 48D color histogram + category filter, tuned across 300 Optuna trials — surprising finding: the 48D histogram outperformed a 2048D ResNet50. Ran 27 configurations across CLIP, DINOv2, ResNet50, and EfficientNet; the category filter alone gave +6.9pp R@1 with zero new features and never made any query worse.
